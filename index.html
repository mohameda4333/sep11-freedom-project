Mini-Shooter Game

About my Game

This is a simple shooting game that I made using kaboomjs. The user has to use the "player sprite" to shoot "bullet" sprites at the "enemy" sprites. Every successful shot should increase your score up by one.

How to Play

Use the w/a/s/d keys to move your player sprite around the screen. Press the space bar to shoot bullets. Hit as many enemy sprites as possible before they leave the screen.

Game info

  • player movement using w/a/s/d keys
  • Shooting function using the space bar
  • Enemy sprites that move across the screen
  • Score functoin that increases when you hit the enemy

What I Learned

While creating this game, I learned how to use kaboomjs to build a basic game. I practiced creating sprites, moving objects, and detecting collisions. I used Claude.ai to help me understand how the collision function works and how to use it in kaboomjs since it was new to me.

Future edits beyond the MVP
